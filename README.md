1. Project Overview
This project focused on analyzing an E-commerce dataset. The main goal was to organize raw data into a structured format to enable effective business insights, such as calculating total revenue and identifying customer buying patterns using SQL.
2. Methodology
To achieve the objectives, we followed these steps:
• Data Preparation: we cleaned the raw data and created a relational model consisting of three main tables: Customers, Products, and Orders.
• Data Analysis: we used SQL queries to perform data aggregation and filtering. This allowed me to calculate key metrics, such as total sales per region and product performance.
• Documentation: we used GitHub to maintain version control of our code and stored the visual results (screenshots) of the SQL queries in a dedicated outputs folder.
3. Key Learnings
Through this task, we gained practical experience in:
• SQL Proficiency: Improving our ability to join tables and perform complex data extractions.
• Data Integrity: Understanding the importance of cleaning data before analysis to ensure accurate reporting.
• Workflow Management: Learning how to document projects professionally using GitHub, which is an essential skill for any Data Science career.
